Getting Started

At first, you need to insert code in the Head part of your html file.

<script defer type="text/javascript" src="memo.js"></script>
<link rel="stylesheet" type="text/css" href="./style.css"></link>

To create a new memo, you need to have a div tag with a unique Id in the Body part of your html file.
Here is example:

<div id="example-1"></div>

JS Object Explaination - Memo Object

Attribute Type Description Default Value Note
location String The position you want to insert this memo in html undefined You must provide this attribute as a parameter in the constructor. See Example Below.
rowNum Integer The number of rows of the memo 0 This attribute will calculate and update automatically.
id String Unique id for this memo undefined You must provide this attribute as a parameter in the constructor. See Example Below.
title String Title for this memo undefined You must provide this attribute as a parameter in the constructor. See Example Below.
data Array includes Dictionary only The content you want to display in this memo table [] You must provide this attribute as a parameter in the constructor. See Example Below. In addition, Dictonaries must all have the same keys and these keys must be the same as all category you provided plus "id" (unique id).
maxRow Integer Maximun number of Rows -1 You will modify this value by calling setMaximumRow()
categories Array includes String only Categories for the memo, which decides how many columns this memo will have. [] You must provide this attribute as a parameter in the constructor. See Example Below.

Constructor

createMemo(location, id, title, category, data, maxRow)

This function will create a Default memo based on given parameters.
location: String, the position you want to insert in html. In the previous example, "example-1" will be location
id: String, unique id for this memo
title: String, title for this memo
category: Array includes String only, categories for the memo, which decides how many columns this memo will have
data: Array includes Dictionary only which Dictonaries must all have the same keys and these keys must be the same as category plus "id" (unique id), the content you want to display in memo

const example1 = new memo();
example_1.createMemo('example-1', 0, 'sample memo-0', ['Row', 'Notice', 'Deadline', 'Order Adjust'], [["Need to work at 9", '2022-03-10'], ["Need to work on personal", "2003-03-11"]], 100);

Common Methods

insertdata(data)

This function will insert new Row(s) in memo based on given data.
data: Array includes Dictionary only which Dictonaries must all have the same keys and these keys must be the same as category plus "id" (unique id)

example_1.insertdata([["drink", '2000-03-20'], ["eat", '2011-03-20'], ["sweep floor", '2011-06-17']]);

deleteRow(rowNum)

This function will delete a Row in memo based on given id.
rowNum: Integer, row number in memo, which is stored in dictionary mentioned above

example_1.deleteRow(1);

deleteCategory(category)

This function will delete a Category in memo based on given name.

category: String, the name of target Category

example_1.deleteCategory("DEADLINE");

setMaxRow(maxRow)

This function will set a limit on the maximum number of rows
maxRow: Positive Integer, the maximum number of rows

example_1.setMaxRow(100)

setHeaderColor(color)

This function can change the header colour of memo based on given color.
color: String, new colour (must be color).

example_1.setHeaderColor("purple")

setRowColor(rowNum, color)

This function can change the row colour of memo based on given row number and color.
rowNum: integer, row number.
color: String, new colour (must be color).

example_1.setRowColor(1, "blue")

setBackgroundColor(color)

This function can change the background colour of Table based on given color.
color: String, new colour (must be color).

example_1.setBackgroundColor("blue")

addCategory(category)

This function can cadd catgeory(uppercase) based on the given category string, insert before Order Adjust
category: String, category name.

example_1.addCategory("test")

deleteRankButton()

This function can delete the rank button

example_1.deletRankButton()

editNotice(rowNum, newContent)

This function can edit the notice content based on the given row number and new content

example_1.editNotice(2, 'test')
